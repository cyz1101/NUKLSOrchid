嘉德利亞蘭

過去與現在之演變

嘉德利亞蘭(Cattleya)為複莖類蘭科植物，栽培容易，生長溫度範圍較大，在5℃ ~35℃之間。在台灣獨特的海洋性氣候，冬天不會低到0℃，夏天也不會超過40℃，不像大陸型氣候，溫度有太低或太高的情形，所以全省均可栽培嘉德利亞蘭。

嘉德利亞蘭花朵很大，色彩鮮艷，頗有王者風範，故稱「蘭花之王」，很早就被引進栽培及育種。深得世界各地趣味栽培者的喜好，在世界各地的蘭展均佔有一席之地。

台灣在日據時代，即被引進栽培，當時因組織培養技術尚未成熟，因此都以傳統分株方式來繁殖，由於繁殖速度慢以及來源有限，價格十分昂貴，只有少數有錢人如醫生能栽培欣賞。光復後，留在台灣的品種陸續被很多醫生及專家作為育種材料，育成許多新的品系如美齡蘭、鹿港蘭等。

1970年王博仁博士回到中研院，開始進行有關蘭花的組織培養研究，陸續有許多學者參與研究並教導學生相關技術。由於國民所得提高，蘭花組培繁殖能力的成長不如需求的旺盛，形成供不應求的情形，此階段為台灣蘭花發展史上的黃金時代。

1980-1990年台灣各地都成立了蘭藝協會及蘭花相關組織，推動嘉德利亞蘭的栽培與欣賞，幾乎每個假日都有蘭展的活動，使得嘉德利亞蘭的價格水漲船高，好的實生苗一株可賣1,000~3,000元，得獎的優良銘花，售價更高達30-50萬元，因此大家競相投資嘉德利亞蘭的生產，少則100-200萬元，多則上千萬元。由於一窩蜂的投入，產量激增，而當時外銷巿場小，以致在1990年市場崩潰，很多嘉德利亞蘭的栽培場成為廢園，損失極大。

1990年以後，台灣嘉德利亞蘭生產進入重新整頓的階段，近十年來台灣蘭花界積極參加世界各地的蘭展，創造大量的嘉德利亞蘭商機，使得嘉德利亞蘭的產銷趨於穩定，一些開花性不佳的品種，都被淘汰掉，取代而來的是栽培容易，開花性佳，品質穩定之優良商業品種。

隨著國際間交流日益頻繁，台灣大量引進國外新品種來繁殖栽培，同時積極進行育種，因此育成不少國際知名的品種，如深黃花的「永康2號」(Blc.HWa Yuan Gold "Y.K#2")，得到加拿大溫哥華世界蘭展冠軍，也榮獲英國皇家園藝協會的第一獎(FCC)。

產品特性

嘉德利亞蘭大多原產於中南美洲，北起自墨西哥，南至巴西、巴拉圭一帶的熱帶雨林中，少數產於海拔600-1,800公尺的高山上。當地雨季及乾季十分明顯，年均溫介於15-25℃，日夜溫差頗大，在7-10℃之間。

嘉德利亞蘭近緣屬很多，如布拉索蘭屬 (Brassavola) 蕾利亞蘭屬 (Laelia)、 索非亞蘭屬 (Sophronitis)、樹蘭屬(Epidndrum)以及扁唇蘭屬 (Diacrium)等，由於值株形態近似，相關屬的原生種達數百種之多，十分容易進行屬間雜交(圖6)，例如目前商業栽培的Blc. Slc. Pot. Eplc.等皆是人工雜交屬。

嘉德利亞蘭在長日下進行營養生長，在短日低溫下進行生殖生長。也有少數種類為不定期花，花期較難控制。一般定期開花品種，如春天開花的Lc. Irene Finney"York"，夏天開花的Lc.Joseph Hampton"Pagent"，秋天開花的Lc. George Boldwin以及冬天開花的品種如Lc. Drumbeat及Blc. Emily Shupe "Grande"等，均可經由每天電照4小時以延遲開花60-70天，達到產期調節的目的(圖7)。

嘉德利亞蘭相關屬蘭花的瓶苗、苗株以及開花株各個階段，均可作為商品出售熱帶地區業者大多進口瓶苗或小苗，以降低成本，而溫帶地區業者大多進口成 株或開花株直接銷售，減少長時間的栽培成本。

一般交配育種出來的苗株經2-4年栽培後即可開花，經選拔的優良品種，利用生長點進行組織培養，經18-24個月的培養可獲得大量6~1Ocm的小苗，先種植於15寸(4.5cm)的軟盆內，經4-6個月生長，高約15-2Ocm後，換到2.5寸(7.5cm)軟盆內，經6-8月生長後，高約25-30cm，再換到3.5寸(10.5cm)的盆子，栽培6-12個月即為開花株。以上所述的軟盆規格均為目前最新的水草栽培方法，若使用蛇木屑或樹皮為材料，一般開花株大多使用5寸(15cm)的盆子。 未銷售的可採用分株法再繁殖。

一般商業栽培嘉德利亞蘭，主要以組織分生苗為主，交配種主要供育種之用。瓶苗的代工費用，交配種為70元/瓶，組織分生瓶苗前100瓶代工費用3萬5千元，即350元/瓶。100瓶以後為100元/瓶，生產更多則代工費用愈低。每瓶苗數為30苗。嘉德利亞蘭的生產目前已有專業的瓶苗代工中心，出瓶的小苗有育苗場以及關花株的生產場。由於分工更細更專業，可降低生產成本及提高產品品質。

嘉德利亞蘭一般栽培於遮蔭網下，設備成本不高，加上台灣得天獨厚的海洋性氣候，極具競爭潛力。

出口地區與市場需求

嘉德利亞蘭可作為切花及盆花使用，日本1998年嘉德利亞蘭切花銷售量達 502.8萬支，盆花銷售量57.3萬盆(表4)。由於日本地處溫帶地區，栽培不易，栽培期長，生產成本高，因此台灣外銷應可再擴大。美國地區幅員廣大，在夏威夷、佛羅里達州和加州等地，氣候溫和，栽培蘭花容易，很多趣味栽培者特別喜歡種植嘉德利亞蘭，市場上的流通量大，其需求量應該會大於日本。

歐洲地區荷蘭目前商業栽培嘉德利亞蘭仍然不多，但已育出很多耐低溫、易開花的品種，未來歐洲巿埸應會逐漸擴大。


亞洲除日本外，韓國、香港、新加坡、中國大陸、菲律賓以及泰國。嘉德利亞蘭日漸普及，切花及盆花的需求將日漸擴大。

由於嘉德利亞花期較短，因此台灣最適當的方式是供應種苗，出口到各國栽植開花後銷售。

生產現況

目前台灣嘉德利亞蘭生產面積約25公頃，主要產地集中於台南縣、彰化縣、台中縣以及高屏一帶，瓶苗以代工生產方式為主，僅少數大蘭園自行量產。每年瓶苗生產量約3萬瓶，外銷瓶苗約1萬瓶，苗株全年產量約45萬株，外銷約為35萬株。台灣業者每年參加很多地區的蘭展活動，而台灣的嘉德利亞蘭不僅品種多，品質佳，價格也極具競爭力，帶動了整個嘉德利亞蘭的發展。

早期嘉德利亞蘭均以蛇木屑栽培，近年由於森林的砍伐，蛇木逐漸減少，因此有些業者改用樹皮種植。以上兩種栽植材料均不易保水，須要經常澆水，增加工作負擔，所以逐漸改種水草，不僅容易澆水又省工、生長快，作為組合盆栽使用也比較不易凋謝，因此大大的提高嘉德利亞蘭的品質。

外銷優勢

(1)台灣的嘉德利亞蘭品種大多為自行育種，不論在大型種、中型種或小型種均頗具特色，深得國外買主的喜愛。

(2)國內組織培養代工技術成熟，價格合理，同時國內嘉德利亞蘭生產分工頗細，專業性高，可降低生產成本且保障品質。

(3)台灣屬海洋性氣候，十分適合嘉德利亞蘭的生長，因此台灣的栽培方式十分簡單，生產成本不高，具有競爭力。

(4)台灣專業性蘭園及趣味栽培者約佰200戶，栽培區域接近，形成聚落效應，容易互相觀摩學習及資訊流通。

(5)政府進出口相關單位如國貿局、防檢局與金融服務業文件核發迅速，國內運輸以及航空業發達，利於出口。

外銷劣勢

(1)資訊缺乏，蘭園往往各自為政，以致經常造成生產與行銷不能協調，導致花農的損失。

(2)一般蘭園面積小，無法進行機械化或自動化來降低生產成本，台灣工資又較鄰近國家為高，因此降低出口競爭力。

(3)台灣蘭花組織培養發達，但各家技術不一因而出現許多組培苗變異及病毒感染的現象，影響台灣蘭花種苗的品質及信譽。

(4)台灣位處颱風地區，經常會碰到豪雨與強風，導致蘭園損失，增加生產成本及風險。

(5)雖然國內育成許多新品種，但由於沒有種苗專利保護，無法保障育種者的權利，同時也失去許多商機。

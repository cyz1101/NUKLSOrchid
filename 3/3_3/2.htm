嘉德利亞蘭

  • 目前產業狀況
  • 加強發展外銷之作法

市場資訊

農委會及經濟部應編列經費委託政府及民間駐外單位，每年定期收集國外蘭花市場資訊，提供國內業者參考。

品種改良

(1)加強新品種的開發，以擴大市場的佔有率。

(2)品種權利保護，應比照鄰國日本，儘快公告種苗專利法，保障育種者在國內享有專利的保護。同時由政府透過公會或協會，協助育種者將優良品種在美國、日本及歐洲等世界主要消費市場申請品種權利保護，以擴大國外市場，並保障台灣育種者的權利，才能提供國內育種的誘因，鼓勵業者積極育種。提高台灣的國際競爭力。

栽培管理

加強試驗研究，制定標準的栽培模式，使花農生產的品質穩定並降低生產成 本，提高台灣產品的形象。

經營管理

發展中衛體系(請參見又心蘭及相關屬盆花)以降低生產成本，提高品質及強化銷售管道。

研究發展

整合國內試驗研究機關與學術機構，共同針對嘉德利亞蘭相關屬從瓶苗、小苗到成株不同階段的相關研究，解決瓶苗的變異性，應用生物技術來延長花期、增加花色及花型，提高栽培管理，病蟲害防治及盆花切花貯運技術。

拓展外銷市場

﹙1﹚參與國外蘭展，在國內舉辦大型蘭展打開國內消費巿場並廣邀國外業者來台採購，協助業者向國外促銷。

﹙2﹚鼓勵及協助業者到國外設場，提高國外市場銷售能力。

改善外銷障礙

解決我國輸美水草問題(請參見蝴蝶蘭)，以利帶盆出口，提高巿場競爭力。

設立蘭花事業園區

由政府結合相關單位，依業者之需求，規劃花卉專業園區，出租給業者，以擴大生產規模 (請參閱文心蘭及相關屬盆花)

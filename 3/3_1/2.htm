蝴蝶蘭

  • 目前產業狀況
  • 加強發展外銷之作法

市場資訊

(1)加強巿場資訊收集分析整理，以而場需求引導生產，良好的產銷配合，才能創造更多營利 。

(2)農委會及經濟部應編列經費，委託政府及民間駐外國單位，每年定期收集蘭花巿場資訊，提供生產業者隨時調整規劃生產策略。

品種改良

(1)不斷推出新品極是花卉產業永續經營及保持巿場競爭力的首要命脈，育種者活絡結合產業栽培場的栽培選育而不斷推出新品種是台灣蝴蝶蘭產業具國際巿場競爭力之主要因素之一，多年來由於缺乏品種專利權的保權而致趣味育種逐者逐漸稀少；另外多數蝴蝶蘭業者經營面積不大，沒有長期進行品種改良的能，也致使台灣逐漸喪失蝴蝶蘭產業的品種優勢。

(2)農委會於 2002 年 7 月公告蝴蝶蘭種苗法將可提升台灣蝴蝶蘭競爭力，但政府機構仍需資助公會或協助成立專責業務單位，以協助育種者在西歐、美國、日本等主要蘭苗進囗國家申請新品種專利，保障育種者權利，使活絡的趣味育種者能夠生存及繼續育成新品種；另外農試所花卉研究中心應設立蝴蝶蘭育種研究室，如同過去水稻育種從事長期收集保存種源，並結合民間企業從事，種改良工作。

栽培管理

(1)政府各改良場加強就地協助輔導花農之栽培管理及病蟲害防治技術來提昇蘭苗品質，生產業者必需維持一樣且穩定的品質，貿易商才容易促銷。

(2)需以新品種分生苗為主力，才容易建立栽培管理方法，如此，才能供應高品質、高價位的巿場。

經營管理

(1)每家生產者的溫室面積太小，種植品種太多，產銷失衡，缺乏為農產品外銷而設立之運輸航業等問題已令台灣蝴蝶蘭產業在國際舞台出現市場競爭的危機。

(2)政府需輔導花農共同產銷，例如規劃花卉外銷軟硬體週邊設備，降低花卉外銷運費等

(3)農委會應委託縣巿政府或農會整合業者擴大經營規模及生產管理方式，必要時可以政策性及策略性責成中央政府，籌設蘭花產銷專業區或蘭花生技園區，解決所需土地及資金取得問題 ; 最近中央政府配合台南縣計畫設立「台灣蘭花生物科技園區」，是政府促進蘭花產業昇級及提升其競爭力的英明政策。

研究發展

(1)開發帶花梗苗株新產品外銷，以往台灣一直以出囗中小苗及成熟苗為主，送到國外栽培者從催花到成品出售需 4-6 個月 ( 成熟苗 ) 到 1 年以上 ( 幼苗 ) ，時間很長，而且催花成功率不穩定。以帶花梗成株出囗可以使國外栽培期縮短為 3-4 個月，年週轉率提為 2-3 倍以上，催花率低的問題亦可有效解決。

(2)政府有關部門 ( 農委會及國科會 ) 應持續將蝴蝶蘭等蘭花產業列為政府輔導之重點農業，編列經費結合各學研單位加強對產業所需之技術進行研發。

(3)巿場流通之現有重要品種，政府委託學術機構建立最適宜栽培方法，如此才能有效降低生產成本。

(4)加強分生繁殖技術之研究，解決分生苗變異偵測問題及降低分生苗變異與生產成本的技術。

(5)應用分子生物學等新科技加速育成新品種。

(6)投入蘭苗貯運遇不良環境遭受損傷之防止技術等相關研發工作，使蝴蝶蘭產業能維持競爭力。

改善外銷障礙

美國及加拿大等北美國家，要求進口之蘭苗必需除去水草等栽培介質而裸根包裝，但裸根蘭苗到達客戶蘭場需再回種而增加成本，且須增加 1-2 個月恢復期，並降低蘭花品質。建議政府積極與美方諮商談判，加速解決帶水草蘭苗准許輸入美國及加拿大的談判問題，使台灣蘭苖在北美國家更具競爭力。

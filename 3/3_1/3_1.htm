過去與現在之演變

台灣栽培蝴蝶蘭之演變可明顯的分為三個階段， 1970 年以前只有趣味者栽培原生種，少有雜交育種行為，山採原生種台灣阿媽曾大量流入日本栽培者 ;1971 年至 1980 年趣味栽培者增多，並自國外引進優良栽培種開始嘗試雜交育種工作，育種目標多注重於白花之花型及花徑之改良，育成之瓶苗或蘭苗亦有零星交易 ;1981 年到 1990 年趣味栽培者開始轉型成為專業蘭園栽培者，如兄弟蘭園、石原蘭園、明和蘭園、台大蘭園及一心蘭園等，並積極自國外大量引進各種花色之優良親本，育種目標為加強多樣化花色。另外， 1986 年大型企業如台糖公司開始加入研發工作，育種目標除了花型、花色外，特別重視蘭株之栽培性、開花性、抗病性等能降低生產成本之優良品種選育，期能育出符合一般消費者之多樣化花種。

1988 年以前，蝴蝶蘭產業一直是民間趣味者小規模的經營方式。通常是以簡易溫室種植，只限於數十坪到數百坪，巿場規模僅限於趣味者之流通。產品則是由各趣味者嘗試雜交播種所得的瓶苗或是未開花的小、中苗。購買者希望由分離族群中選拔到優良的單株，繁殖後出售給不同蘭園或趣味者。 1988 年台糖公司投入蝴蝶蘭等精農產業，並於台糖研究所成立園藝系，組成蝴蝶蘭研發的團隊，從事育種、栽培、病蟲害、溫室設施、採收後處理等相關的研究。研發目標就是以發展外銷為導向及企業化經營蝴蝶蘭之產業。

十多年來，台糖不僅建立自己的蝴蝶蘭產業，並徹底改變全台灣的蝴蝶蘭經營型態，其重大影響包括 :

(1) 引進荷蘭 Veno 型的玻璃溫室為原型，進一步改進為本土化、規格化的蝴蝶蘭栽培溫室。此種精密性的溫室，夏天透過遮光網、風扇和水牆的運轉可將溫度控制在 30 ℃以下，冬天透過加溫及保溫設備可將溫度提高到 22 ℃以上；經由本土化的台糖環控一號電腦出動環控設備，可將溫度、濕度和光照控制在適宜蝴蝶蘭生長的環境內。

(2)商品型態由初期設定的切花轉為中、大苗外銷，剛好結合亞熱帶地區高溫生產苗株，溫帶地區涼溫誘導開花、銷售的特殊產業結構，形成互補共生的策略聯盟此舉結合兩地各尚特有的環境優勢，增加雙方的溫室週轉率，且避開切花不易包裝運銷的問題。

(3)選定水草為蝴蝶蘭的栽培介質，使管理過程單純化、省工化，並且使帶盆外銷成為可能，使盆苗送到輸入國後苖株恢復期縮短，提早蘭苗開花。

(4)小病蟲害及病毐病害有效控制，尤其是軟腐病的藥劑防治成效良好。

(5)蝴蝶蘭品種改良的成效優異，各生產業者以自我育成的品種創立品牌如兄弟 (Brother) 、 一心 (I-shin) 、 台糖 (Taisuco) 等，尤其是 Taisuco 白花蝴蝶蘭聲名遠播。

(6)開發蝴蝶蘭花期調節技術，使其能周年開花。

台糖公司建立其企業化經營的蝴蝶蘭產業後，除能大量外銷苗株賺取外匯外，也對台灣蘭花產業產生顯著的影響。首先是國外客戶大量向台灣蘭花業者訂購品質優良的蝴蝶蘭苗，其次是國內巿場逐漸出現價廉物美的蝴蝶蘭盆花，帶動國內蝴蝶蘭盆花市場逢勃發展。舊有趣味性養蘭業者轉型，以台糖公司為師，投入此種蝴蝶蘭企業化的經營模式，興建精密水牆溫室，以水草種植，銷售中苗、大苗，甚至利用台糖的親本，進一步雜交育成自已的雜交種。另外，一些大型企業如陸仕公司、金車飲料公司等發現栽培蝴蝶蘭有利可圖，亦相繼投入蝴蝶蘭產業，使台灣成為全世界蝴蝶蘭苗最大外銷國家。

產品特性

原生蝴蝶蘭約有 50 種，分佈於東南亞熱帶雨林裡之樹幹上，不喜歡直曬陽光，營養生長較適宜的溫度為日溫 28 ℃以下，夜溫 22 ℃以上，成熟蘭株較佳抽梗開花溫度為日溫 25 ℃以下，夜溫 18 ℃維持 12 小時以上。台灣地處亞熱帶區，亦有兩種原生種，即台灣阿媽蝴蝶蘭 ( Phalaenopsis amabilis var.formosa Shimadzu ) 及姬蝴蝶蘭 ( P.equestris (Schauer) Rchb.f. ) 。

瓶苗、蘭苗、盆花及切花皆是蝴蝶蘭產業生產的產品，這些產品可來自雜交的實生苗或優異個體株無性繁殖的分生苗。兩者皆應用植物組織培養技術生產瓶苗，實生苗從雜交播種到出瓶費時 12-16 個月，分生苗從母瓶到大量分生繁殖及出瓶約需 18-24 個月。瓶苗每苗生產成本實生苗約新台幣 3 元，分生苗約 15-35 元素 ( 依單一品系分生繁殖量及困難度而定 ) 。出瓶的蘭苗在環控溫室栽培，不論寶生苗或分生苗其栽培作業方式相同，出瓶苗種在聚合盆或 1.5 寸 (4.5cm) 盆培育 3-5 個月，葉幅至 10-15 公分左右的小苗即可銷售，或換盆於 2.0 或 2.5 寸盆繼續栽培 3-5 個月成為葉幅 20-25 公分之中苗銷售，或繼續換盆於 3.0 、 3.5 或 4.0 寸培育 4-6 個月，成葉幅 30-35 公分的中大苗或大苗銷售。未售出的大苗可放在催花的冷溫環境培育成盆花或切花銷售。

台灣地處亞熱帶很適合蝴蝶蘭苗之生產 ( 圖 1, 於 p.95) ，蘭苗以裸根或帶盆皆很耐運輸 ( 圖 2, 於 p.95) 。因此極有利於在台灣生產蘭苗，運輸到溫帶地區藉其冷涼的氣候開花銷售的產銷結構。

出囗地區與市場需求

瓶苗除少量外銷韓、日、荷蘭等國家外，多數瓶苗在台灣栽培成各階段大小的蘭苗外銷日本、美國等國家 ( 圖 3) 。目前台灣每年出囗苗株數量約為 1,200 萬 ( 日本約 300 萬，美國約 250 萬，大陸約 300 萬，其他地區約 350 萬 ) ，沒外銷的大苗。尤其是紅花品種，在台灣催花後。以盆花或組合盆花在台灣銷售 2000 年蝴蝶蘭盆花已躍界為台灣盆花銷售最大的花卉。台北市每年拍賣量約為 45 萬盆。台灣每年苗株及盆花之產量應有 3,600 萬以上，國際巿場需求量估計約有 7,500 萬 ( 日本巿場 3,000 萬，美國 1,500 萬，歐洲 2,000 萬，大陸等其他地區 1,000 萬 ) 。

日本巿場大部份皆以結合盆花行銷，每年消費約有 323 萬組合盆花，即約有 1,200 萬單株盆花。栽培面積約有 82 公頃 ( 表 2) ，目前每年自台灣進口大株蘭苗約 250-300 萬苗，其餘皆是自行生產，台灣蘭苗生產若具品種品質及價格競爭力，仍能擴增日本巿場。

歐洲巿場的需求逐年增加中，荷蘭是最大供應國， 2000 年約有 840 萬單株盆花銷售量， 2001 年加 19% ，這些盆花多數由拍賣批售，單株盆花每盆平均單價為 5.06 歐元，比 2000 年單價 4.63 歐元高。台灣由於航空運費高，只出囗瓶苗或小苗才有競爭力。

北美巿場大，近年蘭花類盆花已躍昇美國盆花銷售量第一位， 2001 年約有1,212 萬單株盆花銷售量， 1999 年台灣出囗美國蝴蝶蘭約有 200 萬株 ( 約有 500 萬美元 )( 圖 3); 美國蝴蝶蘭盆花產區多集中於美西加州及美南佛州，由於盆花不易運送，美東及中北部大都巿人囗密集區仍有很大而場需求量。

最近中國大陸、韓國、香港亦有極大之需求，尤其是在中國農曆春節前。但中國大陸及韓國亦積極設置蘭苗生產基地，未來可能成為台灣的競爭對手。

生產現況

依據中華盆花發展協會 2001 年 l1 月調查報告，台灣蝴蝶蘭溫室栽培面積年約有 89 公頃 ( 含台糖公司 ) ，有 241 戶栽培業者，年產量約 3,600 萬株，主要產區集中於台南縣、嘉義縣、屏東縣、雲林縣，分別各有 69 戶、 43 戶、 21 戶及 38 戶。四縣栽培面積佔台灣之 80% ，經營類型已朝分工合作轉型，區分專業生產瓶苗埸、專業育苗埸、專業催花埸。

外銷優勢

(1)台灣蝴蝶蘭品種多，趣味育種者活絡，可結合產業栽培埸不斷的推出新品種。

(2)大型企業投入蝴蝶蘭產業，有效的企業管理引導小型蘭園轉型，具有產業聚落效應優勢，可充裕供應國際巿場所需的各種產品及數量。

(3)針對台灣氣候環境，已發展出適宜栽培蝴蝶蘭之亞熱帶型精密溫室，能生產品質穩定的蘭苗，並有政府貸款協助農民興建溫室 。

(4)栽培技術佳，能長期提供一致而優良的品質。

(5)組織培養技術在台灣已成熟，並有專業植物組織培養場將新品種迅速量化生產。

(6)政府核發瀕臨絕種野生動植物保育公約(CITES)文件及檢疫效率高，金融服務業發達，運輸便捷，有利於出口作業。

(7)八成蘭苗以實生苗生產，由於病毐不易經由種子傳染後代，因此較易生產無病毐健康種苗。

(8)由於世界蘭展、亞太蘭展每三年定期在各洲舉辦，使各地消費者了解蝴蝶蘭盆花容易栽培而開花期長之特性，因而逐年擴大全世界巿埸需求量，台灣業者積極參與此方面活動，提昇台灣蘭花產品知名度。

外銷劣勢

(1)多數生產者的溫室面積太小，種植的品種多，不易使用機械化等省工作業來降低生產成本，較難維持穩定蘭苗品質。

(2)蘭苗栽培過程繁雜，不易機悈化而需大量人工，台灣工資遠高於中國大陸及東南亞國家，逐漸喪失競爭力。

(3)多數蘭園需生產兼銷售，對巿場需求不能充分掌握，蘭苗達到成品之栽培期又長，常造成產銷失衡。

(4)缺乏整體產運銷規劃，影響產品國際競爭力。

(5)缺乏新產品，如帶花梗株之外銷尚未充分開發。台灣蝴蝶蘭於冬末春初均有開花株過產之現象，此時溫帶地區消費旺季，以帶花梗株或半開花成株出囗，如日本、大陸、美國與歐洲皆有很大的巿場。

(6)經營規模及管理方式無法擴大及題昇：前者係受制於土地資源取得不易，後者受限於個別業者資金財力不足。

(7)蘭苗為具生命的鮮貨，出囗貯運有別於電子、機器產品，目前台灣尚缺乏為農產品而設的物流運輸航業，使農產品在貯運過程喪失商品價值。

(8)農產品體積大，運輸成本高，單位產值遠低於電子產品，不易爭取到艙位。
